Purpose, Goals and Audience

The purpose of the website is to serve as a informationhub for Realfagskjelleren. Its main goal is to accessibly provide answers to questions students might have about Realfagskjelleren, and secondly also promote Realfagskjelleren and its interests, where the targeted audience of this is mainly students. More specifically Realfagskjelleren wants a quick and simple way to view future events. The primary communication channel for Realfagskjelleren is Facebook and this needs to be reflected on the site. We want if possible to be able to fetch events from their facebook page. While facebook can be great, it is quite cluttered for the user to find the neccecary information. The purpose of the website is to bridge this gap, providing the information to the user in the cleanest matter possible.

Page Layout and Appearance

  • General page layout

    • All our different html pages will be built in a similar fashion. On the right and left side there will be margins each taking 20% of the webpage width. At the top, centered within the margins, we will have a custom made banner for Realfagskjelleren. Directly beneath the banner we have a navbar, also centered within the margins. The remaining place on the page is where the specific content for each site will be. The following mockup image illustrates the general layout. Note the footer containing contact information. This will be the standard footer used throughout the website (even though it's not included in the other mockups).

  • Navbar

    • The navigation bar consists of six elements as shown in the mockup image above. Each element has a black background, orange font color and a border between eachother to visually show the seperation. To show which of the navbar pages we're currently viewing, the background color of the active page in the navbar is made green:

    In addition to a permanent background color on the active page, the color changes when we hover over a non-active page in the navbar:

  • Fonts

    • We decided to go for Georgia as our main font, while our figures and paragraphs use Computer Modern. For both fonts we use Sans-serif as our fallback font. When it comes to font size, we use 40px for the main title, 20px for the navbar, and 14px for the body.

  • Color Scheme and links

    • The official color scheme for Realfagskjellern is orange and black. To somewhat recreate the feeling of visiting Realfagskjelleren we mix in some black and dark colors. Apart from the links used for the navbar, all the links we use will be orange, underlined and change color to blue on hover. The links in the navbar will not have any further visual effects other than what was described in the navbar section.

  • Shadows and Borders

    • In the media section of the website there will be several images and videos displayed in thumbnail size. clicking an image will bring up a larger version of that image, leaving the rest of the gallery in shadow. In addition, every image in the gallery is enclosed in a small border.

Content

  1. index.html - Hjem

    2. This will be the first page that appears when the user visits the page. This will be a clean page where the only content is the header and a short list of future events with a link to the facebook-event, and opening hours. This page can be reached by clicking the logo at the top of the site.

  2. About.html - Om oss

    3. This will include a short paragraph about what Reaflagskjelleren is. This will be divided into a brief historic overview, how to book Realfagskjelleren, and inform about lost and found.

  3. Events.html - Kalender

    4. This page will provide a list of the future events of Realfagskjelleren. While not displayed properly in Figure: 4 we are still working on how to design the list of events.

Javascript Applications

  • NavigationBar.js
    • Especially when viewing the site on mobile the navigation bar script will be important. This will provide a simple dropdown menu that will work as the navigation bar. The dropbown menu will give the user more realstate to view the sites contents.
  • Games
    • games_finn_toeren.js
      • One of the simpler games Realfagskjelleren has to offer. The purpose of this game is to find the one hidden 2 in a standard deck.
    • games_4chan.js
      • A traditional card game based on the infamous website 4chan.
    • games_terning.js
      • A game based on the loaded dice used in Ludøl.
  • events.js
    • This is perhaps the most challenging script to write. We need use Facebook's API to scrobble information about Realfagskjellerens future events.
  • thumbnails.js
    • For the images on Realfagskjellerens media page it is desirable to be able to expand the images. As this task can be handled both by webkit and jQuery we are unsure if Javascript is the tool we will use.
